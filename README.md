# 🎮 PlayFarmScoreboard 사용 설명서

Fabric 클라이언트에서 사용 가능한 HUD 오버레이 모드입니다.  
직접 HUD를 그리거나 배치해서, 서버 정보를 게임 화면에 맞게 커스텀할 수 있습니다.

---

## 🚀 빠른 시작

### 1. 에디터 열기
* **단축키:** `J + K`
* **열리는 조건:** 월드/서버 접속 중 + 다른 화면(인벤토리, 메뉴 등) 미오픈 상태

### 2. 간단 제작 순서
1. `B`(브러시) 또는 `T`(텍스트) 같은 도구 선택
2. 캔버스에서 클릭/드래그로 요소 생성
3. 필요하면 `V`(편집)로 선택 후 이동/크기 조절
4. `닫기` 버튼 또는 `ESC`로 종료하면 **자동 저장**

---

## 🛠️ HUD 에디터 상세 사용법

### 1. 화면 구성
* **상단 바:** 내보내기, 받기, 실행취소, 다시실행, 초기화, 닫기
* **좌측 패널:** 도구 선택 + 레이어 조작
* **우측 패널:** 색상/투명도, 브러시/지우개 크기, HUD 위치 이동
* **중앙 캔버스:** 실제 HUD를 편집하는 작업 공간

### 2. 도구별 사용법 (좌측 패널)

| 도구 | 단축키 | 사용 방법 | 팁 |
| :--- | :---: | :--- | :--- |
| **편집** | `V` | 요소 클릭 선택, 드래그 이동, 핸들 드래그로 크기 조절 | `Shift + 클릭`으로 다중 선택 |
| **브러시** | `B` | 마우스 드래그로 자유 그리기 | `[` `]`로 브러시 크기 조절 |
| **페인트** | `G` | 클릭 즉시 캔버스 전체 채우기 | 배경색 빠르게 만들 때 사용 |
| **사각형** | `R` | 드래그 시작점~끝점으로 사각형 생성 | 색상/알파 먼저 고르면 편함 |
| **원** | `O` | 드래그로 원/타원 생성 | 도형도 생성 후 편집 도구로 재조정 가능 |
| **직선** | `L` | 드래그로 직선 생성 | 선 두께는 브러시 크기 영향 |
| **이미지** | `I` | 이미지 준비 후 캔버스 클릭으로 배치 | 파일 드래그 후 바로 사용 가능 |
| **텍스트** | `T` | 클릭해서 텍스트 생성/편집 | `Enter`로 편집 종료 |
| **지우개** | `E` | 드래그하며 지우기 | `[` `]`로 지우개 크기 조절 |

### 3. 주요 편집 조작
* **다중 선택:** `Shift + 클릭` 또는 빈 곳 드래그(영역 선택)
* **이동:** 선택 후 드래그 또는 **방향키**(미세 이동), **Shift + 방향키**(빠른 이동)
* **레이어 조작:** 좌측 패널의 `위로`, `아래로`, `맨앞`, `맨뒤` 버튼 사용
* **이미지 추가:** 이미지 파일을 에디터 창으로 직접 드래그 앤 드롭 (GIF 지원)

### 4. HUD 전체 위치/크기 조정
* 우측 **`HUD 위치 이동`** 버튼 클릭 후 HUD 박스를 드래그하여 이동
* **크기 조절:** 키보드 `[` 또는 `-` (**-5%**), `]` 또는 `+` (**+5%**)

### 5. 내보내기 및 받기
* 오른쪽 상단 `내보내기` 버튼 클릭, 파일 이름 변경 및 저장 위치 선택
* 받기는 버튼보다 zip파일을 마크로 끌어서 적용하는걸 추천

## ⌨️ 편집 단축키 모음

* **도구 전환:** `V`, `B`, `G`, `R`, `O`, `L`, `I`, `T`, `E`
* **브러시 크기:** `[` , `]`
* **선택 관련:** `Shift + 클릭`, `Ctrl + A` (전체 선택)
* **삭제:** `삭제하기` 또는 `Backspace`
* **작업 제어:** `Ctrl + Z` (실행취소), `Ctrl + Y` (다시 실행)
* **클립보드:** `Ctrl + C / X / V`
* **종료:** `ESC`

---

## 갱신안네
* 스코어보드 값이 오류가 있거나 이상하면
* `/자동심기` 명령어를 입력하면 자동심기 횟수가 갱신 됩니다.
* `/플라이` 명령어를 입력하면 플라이 횟수가 갱신 됩니다.

## 🛰️ 실시간 데이터 토큰
텍스트 요소에 아래 코드를 입력하면 자동으로 실시간 값으로 치환됩니다.

| 토큰                  | 출력 정보      |
|:--------------------|:-----------|
| `{money}`           | 보유 금액      |
| `{cash}`            | 보유 캐시      |
| `{chat}`            | 현재 채팅 타입   |
| `{channel}`         | 채널         |
| `{fly}`             | 플라이 남은 시간  |
| `{auto plant}`      | 자동심기 남은 횟수 |

---

## 💬 명령어 안내

* **/보스바**: 보스바 표시/숨김 토글 (설정 저장됨)
* **/hud**: 현재 HUD 에디터 GUI 비율 조회
* **/hud <1~8>**: HUD 에디터 GUI 비율 설정 (예: `/hud 3`)
* **/hudgui**: `/hud` 명령어와 동일한 기능을 수행하는 별칭

---

## 📂 설정 저장 경로

* **HUD 레이아웃 및 보스바 상태:** `config/playfarmscoreboard/custom_hud.json`
* **플라이/자동심기 캐시:** `config/playfarmscoreboard/fly_cache.json`

## 🧩 문제 해결
* **에디터가 안 열려요:** 월드 접속 상태인지, 다른 인벤토리나 메뉴 창이 켜져 있는지 확인하세요.
* **가져오기 실패:** ZIP 파일이 내보내기 형식이 맞는지 확인하세요. (가져오기 시 현재 HUD는 덮어씌워집니다.)
